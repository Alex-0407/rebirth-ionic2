## Build this project

You can follow Ionic guide to setup your Ionic environment.

[Installing Ionic: http://ionicframework.com/docs/v2/getting-started/installation/](http://ionicframework.com/docs/v2/getting-started/installation/)

Then run:

```bash
$ npm run dev
```

or 

```bash
$ npm run dev
```

Build project:

```bash
$ npm run android
```

```bash
$ npm run ios
```
